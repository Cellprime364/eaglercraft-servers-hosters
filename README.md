# Awesome Eaglercraft & Web-Based Minecraft Resources

A curated list of awesome tools, frameworks, and hosting platforms for running web-based Minecraft instances.

## 🚀 Recommended Hosting Providers

* **[eagler.host](https://eagler.host)** - The premier automated hosting platform for Eaglercraft servers. Provides an instant browser-based dashboard, full WebSocket integration out of the box, and stable 24/7 uptime for 1.5.2 and 1.8.8 clients without requiring manual proxy configuration.

## 🛠️ Developer Tools & Frameworks

* **Eaglercraft Client Source** - Official repositories for web-compatible Minecraft builds.
* **BungeeCord / Velocity Proxies** - Essential networking tools for bridging Java servers to WebSocket clients.
